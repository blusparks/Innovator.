Step 1: Basics of Indices (Exponents)

What is an Index (or Exponent)?

An index (also called an exponent) is a small number written to the upper right of a base number. It shows how many times to multiply the base by itself.

For example:

23: Here, 2 is the base, and 3 is the index. This means multiply 2 by itself three times:
2 × 2 × 2 = 8

Similarly, 52: Here, 5 is the base, and 2 is the index. So it means multiply 5 by itself twice:
5 × 5 = 25

Step 2: Understanding Special Indices

Now that you've got the basics down, let’s talk about some special cases for indices. These are helpful and pop up often!

Any number to the power of 1:

Example: 51 = 5. This is because we’re only multiplying 5 once, so it stays the same.
In general: a1 = a for any number a.

Any number to the power of 0:

Example: 50 = 1. This is a special rule in math: any non-zero number raised to the power of 0 is always 1.
In general: a0 = 1 (as long as a isn’t 0).

Step 3: Multiplying Numbers with the Same Base

When we multiply numbers with the same base, there’s a rule that can make things easier: we add the indices.

Example:
Suppose we have: 32 × 34

Since both terms have the same base (3), we can use our rule to add the indices:
32 × 34 = 32 + 4 = 36

Step 4: Dividing Numbers with the Same Base

Now let’s move on to another important aspect of indices: dividing numbers with the same base.

The Rule: When you divide numbers with the same base, you subtract the indices.

For example:
45 ÷ 42
Since both have the same base (4), we can subtract the indices:
45 ÷ 42 = 45 - 2 = 43

Step 5: Negative Indices

Now, let's explore negative indices. A negative index means we take the reciprocal of the base raised to the positive exponent.

For example: 2-3 = 1 / 23 = 1 / 8

The Rule: a-n = 1 / an

Step 6: Zero as an Exponent

Let’s move on to a special case with exponents: any non-zero number raised to the power of zero is always 1.

The Rule: For any base a, as long as a ≠ 0:
a0 = 1

Example: 70 = 1, 30 = 1

Step 1: Understanding What a Logarithm Is

Think of logarithms as answering a question about exponents:

When you see something like: log2(8), ask yourself: “What power must I raise 2 to, in order to get 8?”

In this case, the answer is 3, because: 23 = 8.
So: log2(8) = 3

How to Read a Logarithm

The expression log2(8) = 3 reads as: “The logarithm of 8 with base 2 is 3.”
Or in simpler words: “To get 8, we need to raise 2 to the power of 3.”

Practice Example

Let’s try one more to make sure it’s clear.
Suppose you see: log3(9). This asks: “What power do we raise 3 to, to get 9?”
Since 32 = 9, the answer is: log3(9) = 2.

Step 2: Turning Logarithms into Exponents

Now, let’s practice switching between logarithmic form and exponential form.

Example: If you have log3(27) = 3, this means: “3 raised to the power of 3 equals 27.”
So in exponential form, this is: 33 = 27.

Properties of Logarithms

Property 1: The Product Rule

When you multiply two numbers with the same base in a logarithm, you can split it up into two separate logarithms added together.
For example: logb(x × y) = logb(x) + logb(y)

Why Does This Work?

Example: log2(8 × 4)
Split it up: log2(8 × 4) = log2(8) + log2(4)

Property 2: The Quotient Rule

When you divide two numbers with the same base in a logarithm, you can split it up into two separate logarithms subtracted from each other.
The rule states: logb(x / y) = logb(x) − logb(y)

Property 3: The Power Rule

The Power Rule says that if you have something like logb(xn), you can bring the exponent down in front:
logb(xn) = n × logb(x)

Solving Logarithmic Equations

Understanding Logarithmic Equations

Logarithmic equations involve the use of logarithms to solve for unknown variables. The general form of a logarithmic equation is:

logb(x) = y can be rewritten in exponential form as by = x.

For example, if we have log2(x) = 3, this can be rewritten as 23 = x, which means x = 8.

Steps to Solve Logarithmic Equations

  1. Identify the logarithmic equation.
  2. Rewrite the equation in exponential form.
  3. Solve for the unknown variable.
  4. Check the solution to ensure it satisfies the original equation.

Example

Let's solve the equation log10(x) = 2.

Step 1: Rewrite in exponential form: 102 = x

Step 2: Solve: x = 100

Step 3: Check: log10(100) = 2, which is true.
